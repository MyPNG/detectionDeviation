# LLM Deviation Review

## REG-092
- Clause: Art77(1)
- Considered References: None

- Deviation Count: 1

- Deviation 1
  - Type: Data deviation
  - Analysis: Art77(1) grants a right to lodge a complaint if processing infringes "this Regulation" (GDPR). The policy states the right applies if processing infringes "data protection law," which broadens the trigger from GDPR to any data protection law. While not necessarily non-compliant, this is a scope difference. Location specifics (habitual residence, place of work, or alleged infringement) and the "without prejudice" qualifier are omitted, which per instructions are treated as omissions, not deviations.
